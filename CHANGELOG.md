# 更新日志

所有重要的变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.1.3] - 2026-01-29

### 新增
- ✨ 添加 @ 技能自动补全功能，在输入框中输入 @ 时显示技能列表
- ✨ 添加 "插入技能引用" 命令，快速插入 @技能名 到编辑器中
- ✨ 支持通过技能名称或中文描述搜索技能

### 改进
- 🎨 补全列表中显示技能分类和启用状态
- 🎨 已启用的技能在补全列表中优先显示

## [0.1.2] - 2026-01-29

### 新增
- ✨ 添加技能分类设置功能，可以手动设置技能的分类（个人/项目/内置）
- ✨ 支持恢复技能的默认分类

### 改进
- 📝 技能分类设置会持久化保存
- 🎨 改进分类设置的交互体验

## [0.1.1] - 2026-01-29

### 修复
- 🐛 修复递归扫描时跳过 `.claude` 和 `.cursor` 文件夹的问题
- 🐛 修复自定义技能根目录路径扫描不到技能的问题

### 改进
- ✨ 添加调试命令，方便排查扫描问题
- ✨ 添加详细的扫描日志输出
- ✨ 改进错误提示信息

## [0.1.0] - 2026-01-29

### 新增
- ✨ 自动扫描技能文件夹下的所有技能
- ✨ 支持自定义技能文件夹路径
- ✨ 自动将技能文件夹添加到工作区
- ✨ 可视化技能管理界面（树形视图）
- ✨ 技能启用/禁用功能
- ✨ 技能中文描述功能
- ✨ 快速查看技能详情
- ✨ 打开技能文件夹功能
- ✨ 实时监听文件系统变化，自动刷新技能列表
- ✨ 配置选项支持（自动添加到工作区、自定义路径）

### 功能
- 支持扫描 `~/.cursor/skills*` 目录下的技能
- 支持扫描项目级别的 `.cursor/skills` 目录
- 支持自定义技能文件夹路径
- 技能按类别分组显示（个人、项目、内置）
- 技能状态持久化存储
- 中文描述持久化存储
