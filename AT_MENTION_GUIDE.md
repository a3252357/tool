@algorithmic-art# @ 技能引用功能使用指南

## 功能说明

扩展提供了两种方式来引用技能：

### 方法一：自动补全（可能在某些编辑器中不工作）

1. 在编辑器中输入 `@` 符号
2. 应该会自动显示技能列表
3. 继续输入技能名称进行过滤
4. 选择技能后自动插入 `@技能名`

**注意**：由于 VS Code 补全提供器的限制，自动补全可能在某些特殊输入框（如 Cursor 的 AI 对话框）中不工作。

### 方法二：命令插入（推荐，更可靠）

1. 按 `Ctrl+Shift+P` (Windows/Linux) 或 `Cmd+Shift+P` (macOS) 打开命令面板
2. 输入 "Cursor Skills: 插入技能引用"
3. 从列表中选择要引用的技能
4. 技能引用会自动插入到当前编辑器中，或复制到剪贴板

## 使用场景

### 在 Cursor AI 对话框中引用技能

由于 Cursor 的 AI 对话框可能不支持标准的补全提供器，建议使用命令方式：

1. 打开 Cursor 的 AI 对话框
2. 按 `Ctrl+Shift+P` 打开命令面板
3. 选择 "Cursor Skills: 插入技能引用"
4. 选择技能后，`@技能名` 会被复制到剪贴板
5. 在 AI 对话框中按 `Ctrl+V` 粘贴

### 在普通编辑器中引用技能

在普通编辑器中，两种方法都可以使用：

**方法 A：自动补全**
- 直接输入 `@`，应该会显示技能列表

**方法 B：命令插入**
- 使用命令面板插入技能引用

## 故障排除

### 自动补全不工作？

1. **检查扩展是否已激活**
   - 查看输出面板（视图 → 输出 → 选择 "Cursor Skills Manager"）
   - 应该看到 "Cursor Skills Manager 已激活" 和 "技能补全提供器已注册"

2. **检查是否有技能**
   - 使用 "Cursor Skills: 调试：显示扫描信息" 命令
   - 确认有技能被扫描到

3. **手动触发补全**
   - 在编辑器中输入 `@`
   - 按 `Ctrl+Space` (Windows/Linux) 或 `Cmd+Space` (macOS) 手动触发补全

4. **使用命令方式**
   - 如果自动补全不工作，使用 "插入技能引用" 命令更可靠

### 补全列表为空？

1. 检查是否配置了技能路径
2. 使用调试命令查看扫描结果
3. 确认技能文件夹中有 SKILL.md 文件

## 调试

如果遇到问题，可以：

1. **查看控制台日志**
   - 按 `Ctrl+Shift+P` → "Developer: Toggle Developer Tools"
   - 切换到 Console 标签
   - 输入 `@` 时应该看到补全相关的日志

2. **使用测试命令**
   - 按 `Ctrl+Shift+P` → "Cursor Skills: 测试补全功能"
   - 这会手动触发补全

3. **使用调试命令**
   - 按 `Ctrl+Shift+P` → "Cursor Skills: 调试：显示扫描信息"
   - 查看技能扫描情况

## 建议

对于 Cursor 的 AI 对话框，**强烈建议使用命令方式**（"插入技能引用"），因为：
- 更可靠，不依赖补全提供器
- 支持搜索和过滤
- 显示完整的技能信息
- 自动复制到剪贴板，方便粘贴
